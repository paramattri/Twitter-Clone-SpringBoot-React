# Twitter-Clone-SpringBoot-React

## Introduction
Developed a Twitter Clone where user can tweet and perform various functionalities related to tweets.

Functionalities:
  1) User Registration
  2) User Login
  3) Adding a tweet
  4) Updating a tweet
  5) Deleting a tweet
  6) Liking a tweet
  7) Filter Search
